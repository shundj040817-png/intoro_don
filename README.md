# intoro_don
音楽を聴いて何の曲かを当てるゲームを作りました。
制限時間を付けて、問題を４択にしてランダムで出題されるように作りました。
## 使用技術 
- React
- TypeScript
- Next.js
## 詳細
音楽を流し始めたらタイマーが作動するようにしました。
答案者側がやってて飽きないように問題を４択にしました。
シャッフル関数を用いて問題をランダムに出題されるようにしました。
